# OctoFit Tracker Backend (scaffold)

This directory will contain the Django backend for OctoFit Tracker.

Follow the project instructions in `.github/instructions/octofit_tracker_setup_project.instructions.md` to set up the virtual environment and install dependencies.

To create the virtual environment (example):

```bash
python3 -m venv octofit-tracker/backend/venv
source octofit-tracker/backend/venv/bin/activate
pip install -r octofit-tracker/backend/requirements.txt
```
